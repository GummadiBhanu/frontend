Let--Variable

let is a keyword in JavaScript used to declare block-scoped variables. Unlike var, it does not allow redeclaration in the same scope and is safer to use in modern JavaScript.

  • Block Scope
  • Avoids Redeclaration
  • Supports Reassignment
  • Preferred in Loops
  • Preferred in Conditions

Ex:-- 1.) let name = "JS";
console.log(name); // Output: JS
name = "Frontend Dev";
console.log(name); // Output: Frontend Dev

Ex:-- 2.)let in block scope:
let x = 10;
if (true) {
let x = 20; // different x
console.log(x); // Output: 20
}
console.log(x); // Output: 10

Ternary operator

      
The ternary operator in JavaScript is a short-hand conditional operator that evaluates a condition and returns
        one of two values based on whether the condition is true or false.

      

        

          Syntax Pattern
          Meaning
           Example
          Output
        


        

          a > b ? x : y 
          If a > b, return x, else return y
           5 > 3 ? "Yes" : "No"
          "Yes"
        


        

          condition ? trueValue : falseValue 
          General format 
           age >= 18 ? "Adult" : "Minor" 
          "Adult"
        


        

          isLoggedIn ? name : "Guest" 
          Show name if logged in, else "Guest" 
          true ? "John" : "Guest" 
          "John"
        


        

          score >= 40 ? "Pass" : "Fail" 
          Check if score is passing 
          35 >= 40 ? "Pass" : "Fail" 
          "Fail"
        


        

          x ? "Exists" : "Missing" 
          Check if variable is truthy 
          x = 0 ‚Üí 0 ? "Exists" : "Missing"
          "Missing"
        


        

          value === 10 ? "Ten" : value === 20 ? "Twenty" : "Other" 
          Nesting ternaries 
          value = 20 ‚Üí Output: "Twenty"

üìå Syntax:-arrow function

 Immediately Invoked Function Expression (IIFE)

      
An IIFE (Immediately Invoked Function Expression) is a function in JavaScript that is defined and executed
        immediately after its creation.
      
üìå Syntax:-Immediately Invoked Function Expression (IIFE)
        (function() {

        // Code inside the IIFE

        console.log("IIFE executed!");

        })();


        OR with arrow function:

        (() => {

        console.log("Arrow IIFE executed!");

        })();

      
    
        
  • Create private scope Variables inside IIFE cannot be accessed from outside.
    • 
        
  • Avoid global variables Helps prevent conflicts in large codebases.
    • 
        
  • Run code once immediately Useful for setup or initialization code.
    • 
        
  • Execute async logic quickly Great for modules or config loading.
    • 
        
  • Used in Module Pattern Basis for older JavaScript modular design before ES6.
    •

Constructor Function

      
A constructor function in JavaScript is a special type of function used to create and initialize objects.
        It is called using the new keyword, which automatically sets up the context (this) and returns a new object.
      
üìå Syntax:- Constuctor Function

        function Person(name, age) {

        this.name = name;

        this.age = age;

        }

        let user1 = new Person("Alice", 25);

        console.log(user1.name); // Output: Alice

      
    
        
  • Creating multiple objects- Acts like a class to build many similar objects.
    • 
        
  • Initializing object properties- Set default values for each object.
    • 
        
  • Adding methods to objects- Define behavior using this.method.
    • 
        
  • Prototypes and inheritance- Extend behavior using .prototype.
    • 
        
  • Code reusability- Avoid repeating the same structure for each object manually.
    •
