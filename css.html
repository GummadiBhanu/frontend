Pseudo-classes and Pseudo-elements

Pseudo-classes-in CSS are used to define the special states of an element without adding extra classes or IDs in HTML. They allow styling based on user interaction or element position. For example, `:hover` applies styles when a user points to an element with a mouse, `:focus` applies styles when an element like an input box is active.

Pseudo-elements-in CSS are used to style specific parts of an element or insert content without changing the HTML. Common pseudo-elements include `::before` and `::after`, which allow you to add decorative content before or after an element’s actual content. Others like `::first-letter` or `::first-line` target specific portions of text.

  • Hover
  • Focus
  • Visited
  • Active
  • Disabled
  • Before
  • After
  • First-letter
  • First-line
  • Selection

CSS Selectors

CSS Properties are rules used to style and control the appearance, layout, behavior, and position of HTML elements on a webpage.

  • Universal Selector: *
  • Element Selector: p, h1, div
  • Class Selector: .classname
  • ID Selector: #idname
  • Group Selector: h1, p, div
  • Attribute Selector: input[type="text"]
  • Combinators: div > p, div + p, div ~ p

Ex:--On All CSS Selectors

🔹 Universal Selector (*)

Selects all elements on the page.

* {
margin: 0;
padding: 0;
}

🔹Type Selector (Element selector)

Selects all elements of a specific type.
p {
color: blue;
}

🔹 Class Selector (.class)

Selects elements with a specific class.
p> class="highlight">Hell .highlight {
background: yellow;
}

🔹ID Selector (#id)

Selects an element with a specific ID.
h1> id="main-title">Welcome /h1> #main-title {
color: red;
}

🔹Grouping Selector (A, B)

Applies the same style to multiple elements.
h1, h2, p {
font-family: Arial;
}

🔹Attribute Selector

Selects elements based on attributes.
input type="text" />
input[type="text"] {
border: 1px solid blue;
}

Important Questions on CSS

1. What does CSS stand for?

A: CSS stands for "Cascading Style Sheets.

Current official version of CSS is:CSS3


2. How do you include an external CSS file in an HTML document?

A: Use the link> element in the head> section of the HTML document to link the external CSS file.

3. How do you set the font color of an element in CSS?

A: Use the color property to set the font color of an element.

Ex:--p{
color: blue;
}

4. How can you center an element horizontally in CSS?

A: Use the margin property with auto value to horizontally center an element.

Ex:--div {
width: 200px;
margin: 0 auto;
}

5. What is the difference between margin and padding in CSS?

A: Margin is the space outside the border of an element, while padding is the space inside the border of an element.

Ex:--div {
margin: 10px;
padding: 10px;
}
